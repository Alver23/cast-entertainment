## Descripcion

Api Rest Para administrar eventos y registrar usuarios.

## Instalación

```bash
$ npm install
```

## Ejecutar aplicación

```bash
# development
$ npm run start:dev

# watch mode
$ npm run start:dev:watch

## debug mode
$ npm run start:dev:debug

# production mode
$ npm run start
```

## Generar build de la aplicacion

```bash
$ npm run build
```

## Ejecutar Test unitarios
```bash
$ npm test
```
